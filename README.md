# Night Watch – Movie Directory
#### A front-end web application that lets users search, browse, and save their favorite movies using the TMDB API.
<img src="./images/Night-Watcher-Preview.png" alt="Night Watch App Screenshot"/>

# Description

Night Watch is a sleek movie directory app built using HTML, CSS, and JavaScript.
The application fetches real-time movie data from the TMDB API, displays trending movies, allows users to search for any title, view full details (including trailers and streaming providers), and save movies to a personal Favorites list stored in localStorage.

## Features

✔ Trending Movies Section — Automatically loads trending movies of the week
✔ Search Functionality — Search any movie with real-time results
✔ Movie Details Modal — Displays poster, overview, rating, release date, trailer, and streaming providers
✔ Watch Providers Integration — Shows where the movie can be streamed (US region)
✔ Add to Favorites — Saves movies locally using localStorage
✔ Favorites Section — Displays saved movies and keeps them persistent
✔ Responsive Grid Layout — Optimized for various screen sizes
